Information on Base64

Base64 is a text encoding scheme that represents binary data in ASCII string formats. Base64 was originally used in mail transfer because it is hard to be changed by malicious users.
Note that this page will only accept the letters:
"a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z"
and will be converted to lower case

This is because there are 64 characters in the characterset which are common and printable.
